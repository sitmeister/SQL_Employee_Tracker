
# mysql Employee Tracker

![MIT](https://img.shields.io/badge/License-MIT-yellow.svg)
## Description
The purpose of this command-line app is to allow the user to manage a company's employee database. This content management system enables one to view, add, and remove employees. This was my first time manipulating a database through mysql in my own app.
